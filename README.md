# Projeto EduSolo

Projeto submetido ao concurso Agrinho 2026.

EduSolo — Conservação e Produtividade Agrícola Sustentável

Uma plataforma educacional interativa desenvolvida para conscientizar estudantes, professores e a comunidade sobre a importância da conservação do solo para a agricultura sustentável, a segurança alimentar e a preservação ambiental.
O projeto utiliza recursos modernos de desenvolvimento web para transformar conteúdos científicos em uma experiência de aprendizagem dinâmica, acessível e envolvente.

---

## Sobre o Projeto

O EduSolo aborda os principais desafios relacionados à degradação dos solos agrícolas e apresenta soluções sustentáveis utilizadas no Brasil e no mundo.

A plataforma foi desenvolvida com foco em:

Educação ambiental;
Geografia;
Ciências Agrárias;
Sustentabilidade;
Metodologias Ativas de Aprendizagem;
Acessibilidade Digital.

---

## Funcionalidades
🎯 Conteúdo Educativo
Introdução ao tema da conservação do solo;
Diagnóstico dos principais problemas ambientais;
Panorama global da degradação dos solos;
Impactos ambientais, econômicos e sociais;
Casos reais de sucesso;
Estratégias modernas de conservação.

🎮 Simulador Interativo
O usuário assume o papel de gestor de uma propriedade rural e deve tomar decisões para melhorar a saúde do solo.
O sistema fornece:

Feedback imediato;
Consequências das escolhas;
Aprendizagem baseada em problemas.

📑 FAQ Educativo
Sistema de perguntas frequentes com:

Explicações simplificadas;
Conceitos agrícolas importantes;
Navegação intuitiva.

🎠 Carrossel de Casos de Sucesso
Apresenta exemplos de:

Integração Lavoura-Pecuária-Floresta (ILPF);
Plantio Direto;
Recuperação de áreas degradadas.

🎥 Espaço Multimídia
Área destinada à inclusão de:

Vídeos educativos;
Podcasts;
Cartilhas;
Materiais complementares.

## Objetivos Pedagógicos

Compreender os processos de degradação dos solos;

Identificar práticas sustentáveis de manejo;

Relacionar conservação do solo com produção de alimentos;

Desenvolver pensamento crítico sobre sustentabilidade;

Incentivar o uso de tecnologias agrícolas modernas.
